# 🌟 Selva Gaming Portal - Custom Template Three

Welcome to **Selva**, a comprehensive gaming clan ecosystem and platform built upon the responsive architecture of **Template Three** from Elzero Web School. This project marks a major milestone, where a massive web template was completely transformed from its original layout into an immersive, dark-themed **Luminous Purple** gaming hub for Team Selvador.

## 🎨 Project Concept & Visual Identity
The core objective of this project was mastering complex HTML structures and deep CSS styling techniques (such as advanced Flexbox/Grid layouts, precise pseudo-elements, custom transitions, and interactive animations).
* **Vibe:** Modern, sharp, futuristic, and highly competitive esports atmosphere.
* **Color Palette:** Deep Obsidian Blacks, Clean Whites, and vibrant **Luminous Neon Purple (`#a855f7`)** accents.
* **Assets Integration:** Crafted to seamlessly blend with the **Selvador** gaming branding, using strategic layout gaps so that transparent background graphics flow naturally within the design.

---

## ✨ Features & Re-engineered Sections

- **🎮 Dynamic Gaming Layout:** A fully customized responsive structure tailored for esports rosters and gaming stats.
- **🏆 Selvador Championship 2026 (Events):** A redesigned interactive countdown module set up with exciting match descriptions instead of static filler text.
- **💎 Clan Membership Pricing:** Completely re-imagined pricing matrices converted into gaming subscription tiers:
  - **Rookie ($15):** Entry-level access for upcoming recruits.
  - **Elite ($25 - Most Popular):** Includes premium Discord badges, dedicated scrims, and coaching slots.
  - **Champion ($45):** Ultimate VIP tier with custom clan tags and advanced performance analytics.
- **📊 Clan Stats Counter:** Replaced standard corporate counters with elite esports milestones (Active Gamers, Won Tournaments, Global Scrims, Clan Matches).
- **🟢 Brand-Specific UI Elements:** Custom social media and contact grids utilizing precise Font Awesome Solid & Brand icons, complete with glowing hover effects (such as WhatsApp's authentic `#25D366` hue backed by a luminous `drop-shadow` effect).
- **⚡ Advanced Pseudo-Injections:** Bulletproof CSS architecture utilizing custom checkmarks precisely injected via `::before` unicode attributes (`\f00c`).

---

## 🛠️ Technologies Used

* **HTML5:** Clean, semantic markup tailored for modern layouts.
* **CSS3:** Custom properties (variables), grid systems, pseudo-selectors (`::before`, `::after`), fluid media queries, and high-fidelity neon typography shadows.
* **Font Awesome v5/v6:** Solid & Brand vector icon kits.
* **Google Fonts:** Sharp typography curated for digital web experiences.
